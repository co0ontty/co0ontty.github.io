##python3 基础用法
输入字符串以空格分割  
```python
inp = input().split(" ")
```
str 转换 list  
```python
num = list(map(int,str.split(" "))) #利用map函数将以空格分割的str转换成list
```
list 转换 str
```python
str = ','.join(list) #利用join函数将list 转换成以','分割的str
```
list 排序
```python
a = [3,2,5]
a.sort()
print(a)
a = [2,3,5]
```
字典的使用  
```python
a=eval(input())
b=eval(input())
for key,value in b.items():
    if key in a.keys():
        a[key]=a[key]+value
    else:
        a[key]=value
print(a)
```
